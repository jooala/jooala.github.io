Sukeltaja-App UI/UX

This project was done in Software Development course in University of Helsinki. The course is worth of 10 academic credits and takes 4 months to complete.

Client for this project was Sukeltaja Liitto. Our team's main goal was to make the user interface and user experience better for the users of the mobile application as our client requested. We tried to keep all the old features mostly in the same places but make the user experience more visually pleasing and easier to navigate.

Most of the UI and UX design work was done by me. Tools that I used to design user experience included Figma & Photoshop. Frontend is using React Native and Expo and backend is done with Node and Express library. Client was very pleased with the results.
